현재 프로젝트 기준으로 폴더까지 트리로 보면 이렇게 보면 됩니다.

```
cur_gem_mongo_front_260509/
├── .gitignore
├── 1.md
├── package.json
├── package-lock.json
├── src/
│   ├── app.js
│   └── server.js
└── node_modules/          ← npm install 시 자동 생성 (내부는 패키지 파일 다수)
```

- **직접 만든 폴더**: `src`  
- **npm이 만든 폴더**: `node_modules`




스크립트
터미널 명령은 파일로 모아 두거나, 한 번에 여러 요청을 이어서 실행하기 쉽습니다. 예를 들어 test-api.ps1 같은 스크립트에 Invoke-RestMethod 여러 줄을 넣어 두면, 나중에 같은 순서로 다시 실행할 수 있고, CI나 간단한 자동 점검에도 붙이기 좋다는 뜻입니다.
